# DryvIQ

The DryvIQ plugin provides a way to visually manage your DryvIQ operator-managed install in Headlamp. This plugin adds new items to the sidebar for managing parts of the DryvIQ platform as well as new themes for Headlamp.

## DryvIQ Installation

This plugin expects that the DryvIQ Operator is already installed and ready to use in your cluster.

## Plugin Installation in Headlamp for Desktop

Go to the Plugin Catalog, search for the DryvIQ plugin, and click the Install button. Reload the UI (Navigation menu > Reload, or use the notification after installing the plugin) to see the new DryvIQ items in the sidebar.
